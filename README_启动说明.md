# 🚀 快速启动指南

## 一键启动（最简单）

### macOS 用户

**双击运行**：
```
双击 → 启动应用.sh
```

**或在终端运行**：
```bash
./启动应用.sh
```

---

## 首次使用（仅需一次）

### 1. 设置双击运行

右键点击 `启动应用.sh` → 显示简介 → 打开方式 → 选择"终端.app" → 全部更改

### 2. 赋予执行权限

```bash
chmod +x 启动应用.sh
```

- **座位信息**: 使用模拟数据（50个座位）

### 🎯 可用的测试场景

1. **AllSuccess** - 所有座位立即成功（快速测试）
2. **SuccessAtSecond** - 主选失败，备选1成功
3. **SuccessAtThird** - 主选和备选1失败，备选2成功
4. **AllFail** - 所有座位都失败
5. **Random** - 随机结果（60%成功，30%失败，10%异常）

## 常见问题

### Q: 双击后没反应？
A: 检查是否设置了默认打开方式为"终端.app"

### Q: 提示找不到 dotnet？
A: 安装 .NET SDK：`brew install --cask dotnet-sdk`

### Q: 如何切换到真实模式？
A: 编辑 `IGoLibrary.Mac/App.axaml.cs`，设置 `IsSimulationMode = false`
